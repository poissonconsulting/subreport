# subreport: Generate Reports from \`subfoldr2\` Objects

Facilitates generation of .Rmd and .html reports from tables, (code)
blocks and figures (plots and windows) saved using the subfoldr2
package. To quickly view all the results use \`sbr_knit_results()\`. The
\`sbr_n()\` and \`sbr_s()\` functions can be used to insert numbers or
strings saved using subfoldr2.
